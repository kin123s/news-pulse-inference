# 🚀 샘플 워크플로우 (Sample Workflow)

> **안내:** 이 파일은 `.ai/workflows/` 디렉토리 활용을 위한 샘플입니다. 실제 프로젝트의 빌드, 배포, 테스트, 리뷰 등의 반복 작업을 명문화하여 에이전트나 작업자가 일관되게 수행하도록 돕습니다.

## 1. 사전 준비 (Prerequisites)

- [ ] 필요한 환경 변수(`.env`) 셋업 확인
- [ ] 패키지 설치 완료 (`pnpm install` 등)

## 2. 작업 순서 (Steps)

1. **로컬 테스트 스크립트 실행**
   ```bash
   pnpm run test
   ```
2. **빌드 산출물 생성**
   ```bash
   pnpm run build
   ```
3. **결과 확인**
   - 브라우저나 콘솔에서 정상 작동 여부 검증
   - 에러 발생 시 로그 디버깅

## 3. 에이전트 지침 (Agent Instructions)

- 본 워크플로우를 대리로 수행할 때는 각 단계별 성공/실패 여부를 코멘트로 남겨줄 것.
- 빌드 실패 시 자동으로 이전 코드로 롤백하지 말고, 원인을 분석하여 사용자에게 보고할 것.
